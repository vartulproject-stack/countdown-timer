# countdown-timer
Live countdown timer with days, hours, minutes &amp; seconds. Set target date and watch real-time countdown. Built with HTML, CSS &amp; Vanilla JavaScript.
Day 14/100 of #100DaysOfCode. Live countdown timer that takes target date/time and shows remaining days, hours, minutes, seconds in real-time. Displays custom message when countdown ends. Built to practice JavaScript Date objects, setInterval, and time calculations. Tech: HTML5, CSS3, Vanilla JavaScript, Date(), setInterval().
